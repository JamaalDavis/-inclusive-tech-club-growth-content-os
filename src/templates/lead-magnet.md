- Standalone-readable (no assumed context)
- Ordered logically (not randomly)

### If Framework:
- Name each dimension or component clearly
- Provide a short description (2–3 sentences) per component
- Include an example or guiding question per component

### If Canvas:
- Label each section clearly
- Include prompt questions for each section
- Provide fill-in-the-blank structure where useful

## Lead Magnet Accessibility Checklist

- [ ] Document is tagged for screen reader navigation
- [ ] Heading hierarchy is logical (H1 > H2 > H3)
- [ ] All images have alt text
- [ ] Color is not the sole means of conveying information
- [ ] Contrast meets 4.5:1 for body text, 3:1 for large text
- [ ] Font size minimum 12pt in print, 16px equivalent digital
- [ ] Tables have header rows and are not used for layout
- [ ] Links are descriptive, not "click here"
- [ ] PDF exported with accessibility features enabled
